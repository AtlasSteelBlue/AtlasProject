About Airports and Flights

This map shows the size and flight destinations of commercial U.S. airports in 2008.

Libraries

This map is created using the "Data-Driven Document" (D3) library.

Data Import

The D3 library provides importers for CSV (comma separated value) and the Geo JSON formats.

Calculations

The catchment areas of the airports are calculated and visualized by means of Voronoi diagrams. Geographic data is projected on the fly.

Interaction

Roll-over on Voronoi areas shows flight routes from every airport. Voronoi areas can be turned on and off.

Data Source

Public data provided by D3 map example.
