Map Projection

This example demonstrates the projection of shape files on a aerial map.

Software

This map uses the following libraries:

  • RandomEtc Shapefile JS: Reads shape files
  • Proj4 JS: Map projection, converts the Swiss national to the spherical mercator projection system

Data

This map used the following data:

  • Base map: Bingo aerial map
  • Administrative Boundaries: Bundesamt für Statistik, Neuchâtel
