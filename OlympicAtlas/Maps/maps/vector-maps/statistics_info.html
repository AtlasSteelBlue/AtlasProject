About the Statistical Map

This map provides a container for statistical maps that use the same administrative boundaries.

Interaction

This map provides bi-directional interaction between the map and a table. The map will also be centered at the selected table entry.

Categories

Uses an OpenLayers custom control to show the statistical categories.

Libraries

This map builds on OpenLayers's vector capabilities. The map includes the "RandomEtc Shapefile" to import shape and dbf files and the "px" and "Underscore" libraries to directly import PC-Axis database files (a sort of multi-dimensional database file). Also included is a stripped-down version of Colorbrewer to get reasonable color scales.
