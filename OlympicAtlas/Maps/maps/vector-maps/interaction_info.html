About the Interactive Vector Map

This map shows basic interactions with a vector map.

Interaction

This map provides bi-directional interaction between the map and a table. The map will also be centered at the selected table entry.

Libraries

This map build on OpenLayers's vector capabilities. Additionally, the map includes the "RandomEtc Shapefile" library in order to directly import shape and dbf files (actually an improved version as it originally has some flaws).
